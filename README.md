A responsive and interactive frontend web application built using HTML, CSS, and JavaScript.


## Features

- Responsive Layout
- Clean and Modern design
- Interactive elements using JavaScript



## Technologies Used

- HTML5
- CSS3
- JavaScript
## Installation

Follow these steps to set up and view the project locally on your machine:

1. **Clone the repository:**
```bash
  git clone https://github.com/Keerthana-R786/ZYRE.git
```
2. **Navigate to the project directory:**
```bash
  cd ZYRE
```
3. **Open the project in your code editor (like VS Code)**

4. **Run the Project**
- Option 1: Open index.html directly in your browser by double-clicking it.
- Option 2: Use the Live Server extension in Visual Studio Code:
    - Install the extension from the Extensions tab
    - Right-click index.html and choose "Open with Live Server"
    - This will launch your site on a local development server (e.g., http://127.0.0.1:5500/)
